# PLN no Google Cloud — Levantamento de Serviços

  Este repositório contém o levantamento dos serviços de Processamento de Linguagem Natural (PLN)
  disponíveis no Google Cloud Platform, produzido como entrega da atividade do Módulo 6 — Inteli (2026).

  ## Conteúdo

  - [`google_cloud_pln.md`](./google_cloud_pln.md) — documento principal com a descrição de cada serviço,
  exemplos de aplicação via requisição HTTP e comparação com o modelo de análise de sentimento
  desenvolvido no projeto ZAMP.

  ## Serviços cobertos

  1. Cloud Natural Language API
  2. Cloud Translation API
  3. Cloud Speech-to-Text API
  4. Cloud Text-to-Speech API
  5. Dialogflow CX
  6. Vertex AI (Gemini)

  ## Sobre os exemplos

  Todos os exemplos utilizam requisições HTTP diretas, sem bibliotecas específicas do fabricante. Os
  cenários de aplicação são contextualizados ao projeto ZAMP, holding de food service que opera Burger
  King, Subway, Popeyes e Starbucks no Brasil.
